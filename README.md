# Zooma

Joc simplu și rapid de strategie, de tip bubble-shooter, implementat cu ajutorul SFML. Versiunea actuală este v0.4, care implementează majoritatea funcționalităților de bază

## Funcționalități utilizate

* Simularea mișcării șirului de bile pe un traseu
* Tragerea bilelor colorate din proiector
* Proiectorul afișează următoarea bilă care vine și poate să comute între cea actuală și următoarea cu click dreapta
* Detectarea și eliminarea potrivirilor de 3+ bile de aceeași culoare, atunci când sunt trase
* Mesaj la finalul jocului
